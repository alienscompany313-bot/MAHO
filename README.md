# MAHO Enterprise - GitHub Pages + Face ID

این پوشه آماده آپلود در GitHub است.

## روش آپلود

1. در GitHub یک Repository جدید و Public بسازید.
2. تمام محتویات همین پوشه را در ریشه Repository آپلود کنید؛ فایل `index.html` باید در صفحه اول Repository دیده شود.
3. به `Settings > Pages` بروید.
4. در بخش `Build and deployment` گزینه `Deploy from a branch` را انتخاب کنید.
5. Branch را `main` و Folder را `/(root)` انتخاب کرده و Save کنید.
6. پس از فعال‌شدن سایت، آن را با Safari آیفون و از آدرس HTTPS باز کنید.

## فعال‌کردن Face ID

1. نخست با Username و Password وارد MAHO شوید.
2. دکمه `ثبت Face ID` را بزنید و اجازه Face ID را تأیید کنید.
3. خارج شوید.
4. در صفحه ورود، دکمه `ورود با Face ID` را بزنید.

Face ID فقط روی آدرس HTTPS همان GitHub Pages ثبت‌شده کار می‌کند. اگر نام Repository یا دامنه را تغییر دهید، باید Face ID دوباره ثبت شود.

## نصب روی آیفون

در Safari دکمه Share را بزنید و `Add to Home Screen` را انتخاب کنید.
