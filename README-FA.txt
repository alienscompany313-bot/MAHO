نسخه اصلاح‌شده MAHO MMS برای GitHub Pages

فایل‌های داخل این پوشه را مستقیماً در ریشه Repository آپلود کنید.
فایل index.html قبلی را با این فایل جایگزین کنید.
همه فایل‌های PNG نیز باید در همان سطح index.html باشند.
پس از آپلود، چند دقیقه صبر کنید و Safari را Refresh کنید.
اگر نسخه قدیمی دیده شد، سایت را از Home Screen حذف کرده و دوباره Add to Home Screen کنید.
